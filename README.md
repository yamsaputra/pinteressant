5. Wieder anmelden
6. Bilder hochladen klicken
7. Datei auswählen klicken
8. Screenshot mit Bewertung 1,0 hochladen ;) (.png/.svg/.jpeg)
9. Button Auswahl leeren testen
10. Bild wieder auswählen
11. Hinzufügen klicken (Bild wird angezeigt)
12. 6 - 12 wiederholen um mehrere Bilder hochzuladen
13. Auf das Benutzer Symbol klicken
14. Display Name ändern, Theme ändern, Socials hinzufügen (Änderungen werden direkt angezeigt, aber nur übernommen wenn man auf Profil speichern klickt)
15. Default Columns und Default Gap anpassen und gucken wie sich die Anzeige verändert
16. Seite schließen 
17. Seite wieder öffnen (immer noch angemeldet und alles da)
18. Seite schließen und Local Storage leeren
19. Seite wieder öffnen
20. Einloggen (Alles ist noch wie eingestellt)
21. Galerie leeren oder mit dem x einzelne Bilder löschen
